Register MLabs without Credit Card

  • Register an account at mLab.com LINK
  • Create New Database (Image 1)
  • Choose Provider (Image 2) and continue (button is at bottom right)
  • Choose AWS Region that is closer to you.
  • Enter Database Name
  • Choose your database and you should see result like Image 3
  • Proceed by go to Users tab and Add database user (Image 4)
  • REMEMBER THE CREDENTIAL CREATED AT STEP 7. THIS IS YOUR DBUSER AND DB PASSWORD.
  • Copy and Paste the following code snippet into mongoose.js (assuming you are following course structure)
  • Replace the REMOTE_MONGO variable with your link and credentials (image 5)

mongoose.js code snippet

                var mongoose = require('mongoose');

                //copy the link from "To connect using a driver via the standard MongoDB URI" section
                //insert db user name and password here
                const REMOTE_MONGO = 'replace text here with MongoDB URI.';
                const LOCAL_MONGO = 'mongodb://localhost:27017/TodoApp';
                const MONGO_URI = process.env.PORT ? REMOTE_MONGO : LOCAL_MONGO;
                
                mongoose.Promise = global.Promise;
                mongoose.connect(MONGO_URI).then(() => {
                    console.log('Connected to Mongo instance.')
                }, (err) => {
                    console.log('Error connecting to Mongo instance: ', err);
                });
                
                module.export = { mongoose };
